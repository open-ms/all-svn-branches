FAQ

  • MSExperimentExtern crashes with the error message 'file size limit exceeded'

    LSF (large file support) has to be activated eyplicitly in many distributions. Use the configure flag '--with-cxxflags=-D_FILE_OFFSET_BITS=64'.

  • After compiling OpenMS successfully, I receive a linker error saying something about QString:QString(std::char...)

    Try compiling OpenMS with a multithread version of QT. In order to do that use the '-thread' falg in Qt's configure and run OpenMS configure with the option '--with-qt-mt'.
